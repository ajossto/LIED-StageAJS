# Codex Analysis Workspace

Organisation du dossier :

- `papers/` : sources LaTeX et PDF des articles
- `data/` : sorties CSV, JSON et figures par round d'analyse
- `scripts/` : scripts de collecte, de rendu et d'analyse
- `notes/` : mémos de session et briefings
- `reviews/` : reviews textuelles et réponses aux critiques
