# Banque des versions ZIP intéressantes

Ce dossier centralise les archives `.zip` conservées pour futures simulations et comparaisons.

## Archives déplacées

- `claude_nouveau.zip` — v3 hybride (regression)
- `claude3-v3-25-mars-sans-banque.zip` — v3 sans module banque
- `claude3-premier-modele-fonctionnel.zip` — v3 premier modèle fonctionnel (regression)
- `claude_files.zip` (ex-`claude/files.zip`)
- `claude3-v2_codes.zip` (ex-`claude3-v2/src/codes.zip`)
- `claude3-v2_simulation_resultats.zip` (ex-`claude3-v2/src/resultats/simulation.zip`)

## Usage

- Source active de développement : `claude3-v3-27-mars/`.
- Source d'expérimentation historique : extraire une archive ZIP dans un dossier de travail temporaire dédié.
